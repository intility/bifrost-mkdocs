# Buttons

Extension: `attr_list`

Style links as buttons using CSS classes from Material for MkDocs.

## Examples

=== "Result"

    [Primary Button](#){ .md-button .md-button--primary }
    [Secondary Button](#){ .md-button }

=== "Markdown"

    ```markdown
    [Primary Button](#){ .md-button .md-button--primary }
    [Secondary Button](#){ .md-button }
    ```

## With Icons

=== "Result"

    [Get Started :material-arrow-right:](#){ .md-button .md-button--primary }
    [Learn More :material-book-open-variant:](#){ .md-button }

=== "Markdown"

    ```markdown
    [Get Started :material-arrow-right:](#){ .md-button .md-button--primary }
    [Learn More :material-book-open-variant:](#){ .md-button }
    ```
