# Abbreviations

Extension: `abbr`

## Examples

=== "Result"

    The HTML specification is maintained by the W3C. You can write CSS to style your pages and use JS for interactivity. This template uses the MkDocs SSG with the Material theme.

    *[HTML]: Hyper Text Markup Language
    *[W3C]: World Wide Web Consortium
    *[CSS]: Cascading Style Sheets
    *[JS]: JavaScript
    *[SSG]: Static Site Generator

=== "Markdown"

    ```markdown
    The HTML specification is maintained by the W3C.
    You can write CSS to style your pages and use JS
    for interactivity. This template uses the MkDocs
    SSG with the Material theme.

    *[HTML]: Hyper Text Markup Language
    *[W3C]: World Wide Web Consortium
    *[CSS]: Cascading Style Sheets
    *[JS]: JavaScript
    *[SSG]: Static Site Generator
    ```

Hover over HTML, W3C, CSS, JS, or SSG in the result above to see their full forms.

*[HTML]: Hyper Text Markup Language
*[W3C]: World Wide Web Consortium
*[CSS]: Cascading Style Sheets
*[JS]: JavaScript
*[SSG]: Static Site Generator

## Syntax

```markdown
*[ABBR]: Full expansion text
```

Place abbreviation definitions anywhere in the file. Every occurrence of the abbreviated term on the page becomes hoverable.
