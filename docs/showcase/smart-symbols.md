# Smart Symbols

Extension: `pymdownx.smartsymbols`

Smart symbols are automatically converted from plain text sequences into their typographic equivalents.

## Symbol Reference

| Input | Output | Description |
|-------|--------|-------------|
| `(c)` | (c) | Copyright |
| `(tm)` | (tm) | Trademark |
| `(r)` | (r) | Registered |
| `-->` | --> | Right arrow |
| `<--` | <-- | Left arrow |
| `<-->` | <--> | Bidirectional arrow |
| `1/4` | 1/4 | One quarter |
| `1/2` | 1/2 | One half |
| `3/4` | 3/4 | Three quarters |
| `1st` | 1st | Ordinal (first) |
| `2nd` | 2nd | Ordinal (second) |
| `3rd` | 3rd | Ordinal (third) |
